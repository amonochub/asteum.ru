## 🚀 Как начать работу

### 1. Установка (если еще не сделано)

```bash
npm install
```

### 2. Настройка переменных окружения

```bash
cp .env.example .env.local
# Заполните .env.local
```

### 3. Запуск

```bash
npm run dev
# или
make dev
```

### 4. Проверка качества

```bash
npm run validate
npm run test
npm run test:e2e
```
